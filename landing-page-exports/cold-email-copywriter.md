# Description

Most cold emails fail before they're read. Wrong opener, wrong frame, wrong ask. This tool uses Ian's Dyslexic Copywriter voice and the Poke The Bear Hybrid framework to write sequences that feel human, lead with a diagnosis, and ask for a reply — not a sale.

### Features

- No "Hey [Name]" openers — ever
- Poke The Bear Hybrid: stat-as-diagnosis + provocative question + lead magnet offer
- Grade 5–7 reading level — strips AI fluff automatically
- Reply-focused CTAs — the goal is a conversation, not a close
- Write Mode (generate sequences) + Critique Mode (score and rewrite yours)
